## 대응 플레이북 (NIST CSF)
### 대응(Respond)
- 안전모드 진입 → 차량 정지(구동 토픽 차단)
- 침해 세그먼트(도메인) 격리
- 수동 통제로 전환
### 복구(Recover)
- CAN 트레이스 포렌식 → 악성 ID·주입 프레임 식별
- ECU 펌웨어 무결성 검증 + 키 로테이션
### 예방·파훼(Protect)
- CAN MAC(메시지 인증) 추가
- ID 화이트리스트·주기(period) 검사 상시
- 구동계·센서 도메인 분리 + IDS 상주

## 지휘 요약(에이전트)
'CAN 인젝션 (내부버스 구동계 변조)' 확정. 즉시조치: 안전모드 진입 → 차량 정지(구동 토픽 차단). 이후 복구·근본대응(예방)까지 단계 실행. MITRE T0859 (Valid Accounts), T0814 (DoS), T1565 (Data Manipulation).
